# Веб-сайт "W"

## 🛠️ Технологии
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Фреймворк:** Flask 2.3.3
- **Инструменты:** VS Code, Git
- **Хостинг:** Render.com

## 📁 Структура проекта

website/
├── templates/ # HTML-страницы
│ ├── base.html
│ ├── index.html
│ ├── concept.html
│ └── ...
├── images/ # Изображения
├── app.py # Главный файл приложения
├── requirements.txt # Зависимости
└── README.md

## 🚀 Локальный запуск
1. Активируйте виртуальное окружение: `venv\Scripts\activate`
2. Запустите приложение: `python app.py`
3. Откройте: http://127.0.0.1:5000
